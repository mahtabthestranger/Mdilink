**Author:** Al Mamun Oualid

**Features Implemented:**

* F4: View Appointments

Overview
--------

This documentation covers the three doctor-related features implemented for the Medilink Hospital Management System. These features provide secure authentication, appointment management, and medical record handling for doctors.

.. note::
   All features follow secure coding practices including password hashing, SQL injection prevention, and proper session management.
